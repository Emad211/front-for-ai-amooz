# **App Name**: Midnight Canvas

## Core Features:

- Black Screen Display: Display a full-screen black canvas for distraction-free work.
- Timer: A timer function for managing focused work intervals.
- Notes Input: Basic note-taking functionality, displayed as white text on the black screen.

## Style Guidelines:

- Primary color: Dark gray (#333333) to achieve the desired effect. 
- Background color: Black (#000000) for the canvas background.
- Accent color: Light gray (#CCCCCC) for any interactive elements or text to provide contrast and legibility.
- Body and headline font: 'Inter' sans-serif for a modern and readable text.
- Minimalist layout with options for fullscreen mode. Place controls at the edges.
