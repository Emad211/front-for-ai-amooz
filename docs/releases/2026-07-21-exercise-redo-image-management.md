# Exercise redo image management

- Students can now see and remove previously uploaded per-question answer images after a teacher enables redo.
- The displayed image count comes from the server-owned active asset list.
- Removing an image excludes it from the current draft and its next grading fingerprint while preserving finalized attempt history.
- If multiple images remain active for a question, all of them are intentionally read together during OCR and grading.
